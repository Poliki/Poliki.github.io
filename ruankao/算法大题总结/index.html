归并排序

时间复杂度为O(n)

问题描述：

  • 把长度为n的输入序列分成两个长度为n/2的子序列；

  • 对这两个子序列分别采用归并排序；

  • 将两个排序好的子序列合并成一个最终的排序序列

思路：merge函数将两个有序数组合并成一个更大的有序数组；mergeSort函数进行数组的排序；

变量、常量说明：
arr:待排数组
p,q,r:子数组的位置从p到q，另一个子数组的位置从q+1到r
begin：待排数组起始
end：待排数组终止
left：临时存放待合并的子数组
right：同上
n1，n2：子数组长度
ijk：循环变量
mid：临时变量

#include...

void merge(int arr[],int p,int q,int r)
{
 int *left,*right;
 int n1,n2,i,j,k;
 
 n1=q-p+1;
 n2=r-q
 ...
 
 for(i=0;i<n1;i++){
 left[i] = arr[p+1];
 }
 for(i=0;i<n2;i++){
 right[i] =arr[q+i+1];
 }
 
 i=0,j=0;
 
 for(k=p;k<=r;k++){  //(1)
   if(left[i]>right[j]){
       arr[k] = right[j]  //(2)
       j++
   }else{
    arr[k] = left[i];
    i++
   }
 }
}

void mergeSort(int arr[],int begin,int end){
 int mid;
 if(begin<end){ //(3)
   mid =(begin+end)/2;
   mergeSort(arr,begin,mid);
   mergeSort(arr,mid+1,end); //(4)
   merge(arr,begin,mid,end);
 } 
 
}

假币问题

问题描述：

  • 有 n 枚硬币，其中有一枚是假币
  • 己知假币的重量较轻
  • 现只有一个天平，要求用尽量少的比较次数找出这枚假币

思路：分奇偶情况，都是分两边，轻的那边再分两边，奇数也是一样，拿一个出来，如果两边相等那拿出来那个就是假的，否则回归到偶数情况

coins[]: 硬币数组
first ,last：当前考虑的硬币数组中的第一个和最后一个下标

int getCounterfeitCoin(int coins[], int first ,int last){
  int firstSum = 0,lastSum = 0;
  int i;
  
  if ( first==last-1 ) { /*只剩两枚硬币*/
    if(coins[first] < coins[last]){
      return first;
      }
    return last;
  }
  if((last - first + 1) % 2 =0){ /*偶数枚硬币*/
    for(i = first;i <( 1 );i++){ //答案：first+(last-first)/2 从后一个循环找答案
      firstSum+= coins[i];
  }
    for(i = first + (last-first) / 2 + 1;i < last +1;i++){
      lastSum += coins[i];
  }
  if( 2 ){ //答案：firstSum < lastSum
       Return  getCounterfeitCoin(coins,first,first+(last-first)/2;)
  }else{
       Return  getCounterfeitCoin(coins,first+(last-first)/2+1,last;)
  }
}	
   else	{ /*奇数枚硬币*/
      for(i=first;i<first+(last-first)/2;i++){
         firstSum+=coins[i];
      }
      for(i=first+(last-first)/2+1;i<last+1;i++){
         lastSum+=coins[i];
      }
   if (firstSum<lastSum) {
       return  getCounterfeitCoin(coins,first,first+(last-first)/2-1);
   }else if(firstSum>lastSum){
      return  getCounterfeitCoin(coins,first+(last-first)/2-1,last);
   }else{
     return( 3 ) //答案 first+(last-first)/2
     }
    }
  }
}

汉诺塔 Hanoi

问题描述：

  • n个盘子在A处
  • 盘子顺序排列，上面的最小，下面的最大
  • 要把盘子从A搬到C处，可以在B初暂存
  • 但不能出现大盘子压小盘子的情况
void Hanoi(int n,char a,char b,char c) //n个盘子从a通过b移动到c
{
  if(n > 1) {
    Hanno(n-1,a,b,c,d);  // 前n-1个盘子 从a通过c移动到b
    move(n,a,c);         // 第n个盘子，从a移动到c
    Hanoi(n-1,b,a,c);    // 再前n-1个盘子 从b通过a移动到c
  } else {
    move(n,a,c); //递归出口 最后一个盘子 从a到c
  }
}

二分查找 Binary Search

问题描述：

  • 假设表中元素是按升序排列
  • 将表中间位置的关键字与查找关键字比较
  • 如果两者相等，则查找成功；否则利用中间位置将表分成前、后两个子表
  • 如果中间位置记录的关键字大于查找关键字，则进一步查找前一子表，否则进一步查找后一子表
  • 重复以上过程，直到找到满足条件的，使查找成功，或直到子表不存在为止，此时查找不成功
L  :数组名 
ab :数组下标志 
x  :查找的值

2.解题步骤

（2）递归定义最优解的值

LCS最长公共子序列

问题描述：

  • 计算两个字符串x和y的最长公共子串

思路：假设字符串x和字符串y的长度分别为m和n，用数组c的元素c[i][j]记录x中前i个字符和y中前j个字符的最长公共子串的长度。

x，y：长度分别为m和n的字符串。
c[i][j]：记录x中前i字符和y中前j个字符的最长公共子串的长度。
max：x和y的最长公共子串的长度。
maxi，maxj:分别表示x和y的某个最长公共子串的最后一个字符在x和y中的位置 (序号)。

int maxi,maxj;
