汉诺塔 Hanoi

问题描述：n个盘子在A处，盘子顺序排列，上面的最小，下面的最大，要把盘子从A搬到C处，可以在B初暂存，但不能出现大盘子压小盘子的情况

void Hanoi (int n,char a,char b,char c) //n个盘子从a通过b移动到c
{
  if(n > 1) {
    Hanno(n-1,a,b,c,d);  // 前n-1个盘子 从a通过c移动到b
    move(n,a,c);         // 第n个盘子，从a移动到c
    Hanoi(n-1,b,a,c);    // 再前n-1个盘子 从b通过a移动到c
  } else {
    move(n,a,c); //递归出口 最后一个盘子 从a到c
  }
}

斐波那契 Fibonacci

问题描述： 1 1 2 3 5 8 13 21…

F(0)=0，F(1)=1, F(n)=F(n - 1)+F(n - 2)（n ≥ 2，n ∈ N*）

int F(int n)
{
  if(n==0) return 1;
  if(n==1) return 1;
  if(n>1)  return F(n-1)+F(n-2);
}

二分查找 Binary Search

问题描述：首先，假设表中元素是按升序排列，将表中间位置的关键字与查找关键字比较，如果两者相等，则查找成功；否则利用中间位置将表分成前、后两个子表，如果中间位置记录的关键字大于查找关键字，则进一步查找前一子表，否则进一步查找后一子表。重复以上过程，直到找到满足条件的，使查找成功，或直到子表不存在为止，此时查找不成功

变量解释：

L 数组名 ab数组下标志 x查找的值

function Binary_Search(int *L,int a,int b,int x) {
  if (a>b) return（-1）；
  else {
    m = (a+b)/2;
    if (x == L[m]) return (m);
    else if (x>L[m]) return (Binary_Search(L,m+1,b,x));
    else return (Binary_Search(L,a,m-1,x));
  }
}
